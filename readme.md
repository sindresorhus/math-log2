# math-log2

> ES2015 [`Math.log2()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/log2) [ponyfill](https://ponyfill.com)


## Install

```
$ npm install --save math-log2
```


## Usage

```js
var mathLog2 = require('math-log2');

mathLog2(5);
//=> 2.321928094887362
```
